# Day 03 - ALTER TABLE & DELETE (DDL)

## Topics Covered
- ALTER TABLE
- ADD COLUMN
- MODIFY COLUMN
- RENAME COLUMN
- DROP COLUMN
- RENAME TABLE
- DELETE
- UPDATE

## Database
`college_management`

## Tables Used
- teachers
- students

## Operations Performed

### ALTER TABLE
- Added `email` column
- Modified `city` column size
- Renamed `marks` column to `percentage`
- Added `phone` column
- Dropped `email` column

### UPDATE
- Updated email addresses
- Updated phone numbers

### RENAME TABLE
- Renamed `students` to `college_students`
- Renamed `college_students` back to `students`

### DELETE
- Deleted a student record using `DELETE`

## Concepts Learned
- Modifying existing table structure
- Adding new columns
- Renaming columns
- Deleting columns
- Renaming tables
- Updating newly added columns
- Deleting records from a table

## Outcome
Today I learned how to modify an existing table using `ALTER TABLE`, rename tables and columns, update newly added fields, and delete records using `DELETE`. These operations helped me understand how to manage and maintain database tables after they are created.
